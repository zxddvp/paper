一、选题分三个方向：

  • 基础数学
  • 应用数学
  • 中学数学
        选题的同学必须熟悉数学分析、高等代数和解析几何的相关知识、基本思想和基本方法。熟悉中学数学有关问题。

二 、选择毕业论文题目要遵循以下原则

  1. 理论联系实际，注重现实意义
    • 第一，注意选题的实用价值，选择具有现实意义的题目。
    • 第二，要注意选题的理论价值。
  2. 勤于思索，刻意求新
           毕业论文成功与否、质量高低、价值大小，很大程度上取决于文章是否有新意。可以从以下几个方面着眼：
    • 第一， 从观点、题目到材料直至论证方法全是新的。
    • 第二， 以新的材料论证旧的课题，从而提出新的或部分新的观点、新的看法。
    • 第三，以新的角度或新的研究方法重做已有的课题，从而得出全部或部分新观点的文章同样具有新意。
    • 第四，对已有的观点、材料、研究方法提出质疑，虽然没有提出自己新的看法，但能够启发人们重新思考问题。
  3. 知己知彼，轻易适中
    • 第一、知己知己，量力而行
             毕业论文是对学习知识的综合性考核，选题的方向、大小、难易都应与自己的知识积累、分析问题和解决问题的能力，写作经验相适应，要做到“知已知彼”。所谓“知己”，首先，要充分估计到自已的知识储备情况和分析问题的能力。所谓：“知彼”，一是要考虑到是否有资料或资料来源。二是要了解所选课题的研究动态和研究成果，大致掌握写作中可能遇到的困难，以避免盲目性和无效劳动。要注意在已有的研究成果中寻找薄弱环节，即他人研究中存在的疑点、漏洞或不足。有疑点、漏洞的问题，不少是重要的学术论题，以此作为研究的突破口，在理论上修正、补充或丰富已有的结论。只要做到了知己知彼，就能选择一个比较合适的毕业论文题目。
    • 第二、难易适中，大小适度
             要选好毕业论文的题目，把握“适中”的原则是很重要的。
             首先，题目的难易要适中。选题既要有“知难而进”的勇气和信心，又要做到“量力而行”。其次，题目的大小要适度。一般来说宜小不宜大，宜窄不宜宽。题目太大把握不住，考虑难以深入细致，容易泛泛而论。选定小题目，有两种方式，一是直接选个小题目，二是在大题目中选定小的论证角度。角度小，针对性强，容易深入研究。切忌为了“求新”，为了一鸣惊人，就把别人的东西照搬过来，囫囵吞枣，东拼西凑，选题时要引以为戒。

一、应用数学论文题目方向

       应用数学教研室论文题目主要有以下6个方向，每个方向特点以及需要的基本知识如下。
  1. 常微分方程方向
    • 本方向的特点： 常微分方程方向的题目主要取之于常微分方程教材，以常微分方程知识作为基础对某个问题进行探讨，如求方程的解等。
    • 需要的基本知识： 选择本方向的论文题目的学生，首先要熟悉常微分方程的知识、基本方法和基本思想，其次部分题目需要数值分析或者编程的知识。
  2. 代数方向
    • 本方向的特点： 代数方向的论文题目主要取之于矩阵、线性方程组和近世代数等知识，利用这些知识解决问题或者是利用其它知识解决矩阵、线性方程组和近世代数等方面的问题。
    • 需要的基本知识： 选择本方向的论文题目的学生，需要掌握矩阵、线性方程组和近世代数等方面的知识。
  3. 概率与数理统计、金融数学方向
    • 本方向的特点 概率与数理统计、金融数学方向的题目主要以概率统计和金融数学的知识为基础，讨论概率与统计理论的应用和对概率统计理论的深入探讨，而金融数学的题目是讨论数学在经济方面的应用。
    • 需要的基本知识 选择本方向的论文题目的学生，需要掌握概率统计方面的知识，同时对经济方面的相关概念要了解。
  4. 复变函数和实变函数方向
    • 本方向的特点 复变函数和实变函数方向的论文题目主要是对复变函数和实变函数教材的知识进行讨论，一方面加深对这些方面的知识的理解，另一方面把这些知识应用到其它数学分支中，部分题目涉及到泛函分析的知识。
    • 需要的基本知识 选择本方向的论文题目的学生，需要熟悉数学分析、高等代数的知识，掌握复变函数和实变函数的知识。
  5. 运筹与优化方向
    • 本方向的特点 运筹与优化方向的论文题目主要设计运筹与优化在其它数学分支中的应用。
    • 需要的基本知识 选择本方向的论文题目的学生，需要高等代数、运筹与优化方面的知识。
  6. 其它方向
    • 本方向的特点 本方向的题目主要取之于老师的科研方向，对教师科研方向中的小问题进行探讨。
    • 需要的基本知识 根据教师的科研方向确定需要的知识。

二、选论文题目前的准备工作

  1. 确定研究方向： 应用数学方向的论文题目方向多，题目涉及微分方程、复变函数、实变函数、概率统计、近世代数等专业课的知识，因此要做好毕业论文，一定要根据自己的兴趣和擅长选取一个领域作为自己研究方向，把主要力量集中于自己的研究方向上。
  2. 查阅文献搜集整理资料： 根据自己的兴趣和擅长选定题目后，就要去查阅和搜集相关资料，因为做论文都有一定的研究基础，因此你必须了解你选定的方向的研究现状。

三、选题原则

  1. 熟悉论文题目涉及的知识： 毕业论文是对学习知识的综合性考核，选题的方向应与自己的知识积累、分析问题和解决问题的能力，写作经验相适应，否则将无法做该题目的毕业论文。
  2. 对论文题目感兴趣： 兴趣是最好的老师，因此在选题时要注意看对这个题目是否感兴趣，只有对题目感兴趣了，才能做好毕业论文。
  3. 难易适中，大小适度： 要选好毕业论文的题目，把握“适中”的原则是很重要的。 首先，题目的难易要适中。一般来说宜小不宜大，宜窄不宜宽。题目太大把握不住，考虑难以深入细致，容易泛泛而论。选定小题目，有两种方式，一是直接选个小题目，二是在大题目中选定小的论证角度。角度小，针对性强，容易深入研究。切忌为了“求新”，为了一鸣惊人，就把别人的东西照搬过来，囫囵吞枣，东拼西凑，选题时要引以为戒。
  4. 占有较多与论文题目相关的资料： 毕业论文虽然是对学习知识的综合性考核，但也是在前人研究的基础上进行研究，因此一是要考虑到是否有资料或资料来源；二是要了解所选课题的研究动态和研究成果，大致掌握写作中可能遇到的困难，以避免盲目性和无效劳动。所以是否选这个题目取决于能否查阅到或者搜集到与论文题目相关的资料。

  • 特点：信息产业是目前世界极具发展前景的产业之一，其应用遍及工商业、农业、军事、网络和国民经济等多个领域。其中信息熵原理的应用研究更进行得如火如荼，在一些实例中（如网络攻击流量、居民生活消费等）运用信息熵的思想得到了不少具有价值的结论。
  • 需要的基本知识：熟练掌握基本的数学知识；能够较独立进行资料的查询、收集、整理和分析；计算机操作较熟练；有较好的归纳和语言组织能力。
  • 前期准备工作：确定研究方向；做好资料的收集整理工作；潜心研究，力求突破。

二、关于计算科学研究方向

        本方向的内容比较多，也很广泛，主要涉及：
  1. 类聚分析；
  2. DNA计算；
  3. 算法设计和分析；
  4. 数学实验。
  • 特点：
    • 聚类分析是一种重要的统计方法，用统计软件SPSS将其应用到解决实际问题中。处理分类问题的方法主要分为统计方法和非统计方法两大类。其中非统计方法包括：线性规划法、专家系统、最近邻法、遗传算法、神经网络、支持向量机等。
    • DNA计算是目前数据挖掘中一种新兴的计算方法，而基于DNA生化反应的计算机称为DNA计算机，它采用一种完全不同于传统计算机的运算逻辑与存贮方式，在解决某些复杂问题时具有传统计算机无法比拟的优势。DNA计算具有强大的发展潜力和广泛的应用领域，吸引了越来越多的人进行研究。
    • 计算的问题是多种多样的，没有一个好的算法是很难实现其计算问题的，这些计算的问题首先要建立数学模型，然后的关键就是求其解，所以先要对这些模型进行分析，然后给出计算的算法。C++语言目前以成为开发大型系统软件和应用软件必不可少的计算机语言，它是一门可以面向硬件进行编程的语言，所以它的应用价值和前景是非常好的。
    • 数学实验主要是运用学习过的一些数学软件来计算一些复杂的数学模型，在此过程中也培养和训练学生的编程能力。
  • 需要的基本知识：
    • 熟练掌握基本的数学知识，如先修过本专业的某些课程，如《程序设计语言》、《最优化理论》、《C++程序记时计》、《软件工程》等课程。能够较独立进行资料的查询、收集、整理和分析；计算机操作较熟练，熟练掌握数学软件《MATLAB》、《LINDO》、《LINGO》、《MATHEMATIC》中的一种或几种。有较好的归纳和语言组织能力，熟悉简单系统的软件开发和设计，熟练应用一门语言编写和调试程序。
  • 前期准备工作：
    • 确定研究方向；做好资料的收集整理工作；潜心研究，力求突破。
  • 选题时有哪些注意事项：
    • 毕业论文是对学习知识的综合性考核，选题的方向、大小、难易都应与自己的知识积累、分析问题和解决问题的能力，写作经验相适应，要做到“知已知彼”。所谓“知己”，首先，要充分估计到自已的知识储备情况和分析问题的能力。所谓：“知彼”，一是要考虑到是否有资料或资料来源。二是要了解所选课题的研究动态和研究成果，大致掌握写作中可能遇到的困难，以避免盲目性和无效劳动。

一、关于中学数学研究的方向

       关于中学数学研究方向的选题内容十分广泛，主包括下面三个部分。
  1. 中学数学教育研究：
    • 本方向包括中学数学课程和教材研究，中学数学教学方法研究，中学数学思维方法研究，中学数学教育评价研究，高（中）考研究等。
    • 本方向的特点：中学数学教育研究包括中学数学教学研究，应对中学数学课程内容和教材比较熟悉，应掌握一些数学教育学、心理学的理论知识，宜有一定的教育教学经历和经验。
    • 本方向应掌握基本的数学教育理论知识：主要包括初等数学、数学教学论、数学思维论、数学方法论、数学学习论、教育测量与评价理论、初中和高中的数学课程标准等。
    • 本方向需要做的前期准备工作：阅读和研究中学数学教材，学习数学教育学、心理学有关的理论，应查阅近几年一些中学数学期刊，研读有关期刊杂志文章20～30篇，应熟悉近几年的高考数学试题。
  2. 初等数学专题研究：
    • 本方向包括初等数学命题的探索，初等数学定理的推广、移植及应用的研究等。
    • 本方向的特点：此方向的理论性较强，应对初等数学研究和竞赛数学有较深的研究功底。
    • 本方向应掌握基本的数学理论知识：主要包括初等数学研究、竞赛数学、数学分析、高等代数、解析几何等。
    • 本方向需要做的前期准备工作：阅读和研究初等数学研究、竞赛数学，研读有关期刊杂志文章10～20篇，应熟悉近几年的高考数学试题。
  3. 中学数学解题研究：
    • 涉及中学数学解题方法与技巧的课题是很多的，主要包括以下内容：
      1. 数学基本题型解法研究；
      2. 数学问题的一般解法与特殊解法研究；
      3. 解题与证题程序研究，常用解题方法的应用探索；
      4. 错题，错解辩析；
      5. 简化解法与简化证法的研究；
      6. 数学习题的编拟与数学题一题多解的研究，等等。
    • 本方向的特点：应对初等数学和竞赛数学的解题理论有所了解，善于一题多解，会欣赏解题技巧。
    • 本方向应掌握基本的数学理论知识：主要包括初等数学研究、竞赛数学、数学分析、高等代数、解析几何等。
    • 本方向需要做的前期准备工作：阅读和研究初等数学研究、竞赛数学，研读有关期刊杂志文章20～30篇，应熟悉近几年的高考数学试题。

二、撰写论文前的准备工作

        撰写论文前应做好如下准备工作：
  1. 确定研究方向
    • 中学数学教育的内容广阔无边，它可分成许多领域，每个领域又包含无穷无尽的课题，但一般只能深入研究其中若干个领域的一些课题。同学们要想写好论文，最明智的方法是根据自己的兴趣和擅长选取一个领域作为自己研究方向，把主要力量集中于自己的研究方向上。这样做，在容易写好论文。
  2. 做好资料的收集整理工作
    • 写学术论文，要有参考资料，要了解他人的研究成果，要掌握自己的研究方向的动态。这就要求我们做有心人，平时注意购买和阅读有关书籍，经常查阅中学数学杂志上刊登的有关内容，做好有关资料的收集和整理工作。要关注我国中学数学教育新信息、新动态、新趋势，并注意收集这方面的资料。丰富详实的资料积累是撰写数学论文的必不可少的物质基础。
  3. 潜心研究，力求突破
    • 经过一段时间的学习和研究，我们对自己的研究方向有了较深人的了解，这时可结合我们收集整理的资料，选出一个合适的课题，深入了解他人在该课题所做的工作，包括所取得的成果，所采用的方法。研究他人的工作有没有疏漏，结果能否改进，能否有更好的方法，能否采用新的观点，从新的角度去研究此课题。
    • 选题是一项策略性很强的工作，课题选得好是成功的一半。所选的课题，应对当前的中学数学教学与改革有指导意义，应是广大中学数学教师所关心的，对他们的学习和工作是有帮助的。在选题时，既要重视热点，又要注意冷点，有时冷点会变成热点。
